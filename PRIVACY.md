# Privacy Policy — Double Tap Like

_Last updated: June 23, 2026_

**Double Tap Like does not collect, store, transmit, or sell any personal data.**

## What the extension does

Double Tap Like lets you like a video by double-clicking it on X (Twitter),
YouTube Shorts, TikTok, and Instagram Reels. When you double-click a video, the
extension triggers that website's own existing "like" action and shows a brief
heart animation. That is its only function.

## Data we collect

**None.** The extension makes no network requests of its own and has no servers,
no analytics, and no tracking.

## Data stored locally

The only thing the extension saves is your on/off toggle preferences — whether
double-tap-to-like is enabled for each supported site. These settings are stored
locally in your browser using the standard `chrome.storage.sync` API so they
persist between sessions (and sync across your own signed-in Chrome browsers, via
Google's infrastructure, if you have sync enabled). This is configuration data
only. It contains no personal information and is never sent to us or any third
party.

## Permissions

- **storage** — used solely to save the on/off toggle preferences described above.
- **Host access** (x.com, twitter.com, youtube.com, tiktok.com, instagram.com) —
  required so the extension can run on these sites to detect a double-click on a
  video and trigger that site's like button. No other sites are accessed.

## Data sharing

We do not sell or transfer any user data to third parties. We do not use any data
for purposes unrelated to the extension's single purpose. We do not use any data
to determine creditworthiness or for lending purposes.
